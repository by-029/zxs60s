# 走小散每日简报

## 全部执行命令列表

1.设置定时任务：

  /zxs_time 08:00

2.查看所有任务：

  /zxs_doc

3.测试发送：

  /zxs_test

4.激活未激活的晨报任务：

  /zxs_up 

5.删除晨报任务：

  /zxs_doc_del 1

6.取消当前群组定时任务：

  /cl_time

7.群组晨报任务关闭：

  /gg_tasks

------

## 注意事项

- 所有时间使用 Asia/Shanghai 时区（北京时间）

- 定时发送仅在工作日执行

- 支持多群组独立设置不同的发送时间

- 4.5.6任务序号对应 /zxs_doc 命令显示的序号.
